# QR Codes for Terminal Veil

## Sector 3: VEIL-42

Create a QR code with text: `VEIL-42`

### Quick Options:

**Option 1: Online Generator**
1. Go to https://www.qr-code-generator.com
2. Enter text: `VEIL-42`
3. Download and scan

**Option 2: Python (if you have it)**
```python
import qrcode
img = qrcode.make('VEIL-42')
img.save('veil42.png')
```

**Option 3: Use this URL**
```
https://api.qrserver.com/v1/create-qr-code/?size=200x200&data=VEIL-42
```

Just open that URL on your computer, show it to your phone camera, and scan!
